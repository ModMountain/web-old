Billentyűparancsok
Szerkesztési parancsok
Művelet: Windows Mac OS
Visszavonás CTRL + Z ⌘Z
Ismét CTRL + Y ⌘⇧Z
Félkövér CTRL + B ⌘B
Dőlt CTRL + I ⌘I
Aláhúzott CTRL + U ⌘U
Behúzás CTRL + ] ⌘]
Behúzás csökkentése CTRL + [ ⌘[
Hivatkozás hozzáadása CTRL + K ⌘K
Teljes képernyős mód (ki-/bekapcsolás) CTRL + SHIFT + F ⌘⇧F
Súgó menü (megnyitás) CTRL + SHIFT + H ⌃⌥H
Helyi menü (megnyitás) SHIFT + F10 ⇧F10
Kód automatikus kiegészítése CTRL + SPACE ⌃SPACE
*Figyelem: A rendszergazda letilthat bizonyos funkciókat.
