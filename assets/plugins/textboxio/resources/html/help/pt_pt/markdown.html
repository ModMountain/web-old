Formatação com Markdown

Markdown é uma sintaxe para criação de estruturas HTML e formatação sem a necessidade de usar atalhos de teclado ou menus de acesso. Para usar a formatação Markdown, introduza a sintaxe pretendida, seguida pela tecla Enter ou barra de espaços.

Sintaxe de formatação de teclado
Sintaxe Resultado em HTML 
# Maior título
 
<h1> Maior título</h1>
 
## Título maior
 
<h2>Título maior</h2>
 
### Título grande
 
<h3>Título grande</h3>
