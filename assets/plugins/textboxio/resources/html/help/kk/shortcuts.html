Пернетақта пернелері
Редактор пәрмендері
Әрекет Windows Mac OS
Болдырмау CTRL + Z ⌘Z
Қайтару CTRL + Y ⌘⇧Z
Қалың CTRL + B ⌘B
Көлбеу CTRL + I ⌘I
Асты сызылған CTRL + U ⌘U
Шегініс CTRL + ] ⌘]
Шегіністі азайту CTRL + [ ⌘[
Сілтеме қосу CTRL + K ⌘K
Толқын экран режимі (ауыстыру) CTRL + SHIFT + F ⌘⇧F
Анықтама мәзірі (ашық) CTRL + SHIFT + H ⌃⌥H
Контекст мәзірі (ашық) SHIFT + F10 ⇧F10
Кодты авто толтыру CTRL + Бос орын ⌃Бос орын
*Ескертпе: Кейбір мүмкіндіктерді әкімшімен өшіруге болады.
