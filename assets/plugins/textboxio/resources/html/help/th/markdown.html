การกำหนดรูปแบบ Markdown

Markdown เป็นคำสั่งสำหรับสร้างโครงสร้าง HTML และการกำหนดรูปแบบโดยไม่ต้องใช้ shortcuts บนแป้นพิมพ์หรือเมนูในการเข้าถึง ในการใช้รูปแบบ Markdown ให้ใส่ชุดคำสั่งที่ต้องการ ตามด้วยการกดปุ่ม enter หรือปุ่ม space
