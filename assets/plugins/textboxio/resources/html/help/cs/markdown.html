Formátování s pomocí syntaxe markdown

Markdown je syntaxe pro vytváření HTML struktur a pro formátování, při jejímž použití není nutné používat klávesové zkratky ani nabídky. Pokud chcete použít markdown formátování, zadejte požadovanou syntaxi a příslušný výraz ukončete klávesou enter nebo mezerníkem.
