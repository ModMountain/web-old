Formatowanie znacznikami

Znaczniki to elementy składni umożliwiające tworzenie struktur HTML i formatowanie bez potrzeby stosowania skrótów klawiszowych ani uzyskiwania dostępu do menu. Aby zastosować formatowanie znacznikami, należy wprowadzić żądaną składnię, a następnie nacisnąć klawisz Enter lub spację.
