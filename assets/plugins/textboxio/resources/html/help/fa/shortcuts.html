میانبرهای صفحه کلید
فرمانهای ویرایشگر
اقدام Windows Mac OS
لغو کردن CTRL + Z ⌘Z
تکرار کردن CTRL + Y ⌘⇧Z
ضخیم CTRL + B ⌘B
کج CTRL + I ⌘I
زیر خط دار CTRL + U ⌘U
تورفتگی CTRL + ] ⌘]
کاهش تورفتگی CTRL + [ ⌘[
افزودن پیوند CTRL + K ⌘K
حالت تمام صفحه (تغییر) CTRL + SHIFT + F ⌘⇧F
منوی راهنما (باز) CTRL + SHIFT + H ⌃⌥H
منوی زمینه (باز) SHIFT + F10 ⇧F10
تکمیل خودکار کد CTRL + Space ⌃Space
*توجه: سرپرست شما می تواند برخی ویژگیها را غیرفعال کند.
