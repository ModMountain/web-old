עיצוב Markdown

Markdown הוא תחביר ליצירת מבנים ועיצוב של HTML ללא צורך בשימוש בקיצורי מקשים או בגישה לתפריטים. כדי להשתמש בעיצוב Markdown, הזן את התחביר הרצוי ולאחר מכן הקש על ENTER או על מקש הרווח.

תחביר לעיצוב באמצעות המקלדת
תחביר התוצאה ב-HTML
