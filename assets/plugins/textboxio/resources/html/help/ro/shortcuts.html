Comenzi rapide de la tastatură
Editor comenzi
Acţiune Windows Mac OS
Anulare CTRL + Z ⌘Z
Refacere CTRL + Y ⌘⇧Z
Aldin CTRL + B ⌘B
Cursiv CTRL + I ⌘I
Subliniere CTRL + U ⌘U
Indentare CTRL + ] ⌘]
Micşorare indent CTRL + [ ⌘[
Adăugare link CTRL + K ⌘K
Mod ecran complet (Comutare) CTRL + SHIFT + F ⌘⇧F
Meniu Ajutor (Deschidere) CTRL + SHIFT + H ⌃⌥H
Meniu contextual (Deschidere) SHIFT + F10 ⇧F10
Completare automată cod CTRL + SPACE ⌃SPACE
*Notă: Unele funcţii pot fi dezactivate de către administrator.
