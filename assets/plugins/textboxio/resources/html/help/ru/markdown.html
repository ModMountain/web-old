Форматирование Markdown

Markdown — это синтаксическая конструкция разметки текста для создания структур HTML и форматирования без необходимости использовать комбинации клавиш или меню доступа. Для использования Markdown необходимо ввести необходимый синтаксис и нажать ВВОД или пробел.
